# Research

## Description

Gathers and synthesises information on a given topic, producing a concise briefing with sources.

## Triggers

- Manual invocation

## Inputs

| Name  | Type   | Required | Description                              |
| ----- | ------ | -------- | ---------------------------------------- |
| topic | string | yes      | The subject to research                  |
| depth | string | no       | Level of detail (quick, standard, deep)  |
| focus | string | no       | Specific angle or question to prioritise |

## Outputs

A briefing document containing:
- Overview of the topic
- Key findings
- Source references

# Brainstorm Domain Names

> **Generate creative domain names grounded in brand attributes, product purpose, and audience — with availability patterns, scoring, and social strategy.**

## What It Does

This skill generates 20–30 creative domain name candidates using eight naming strategies (compound words, portmanteaus, metaphors, invented words, abbreviations, verb-based, audience-centric, real word modifiers). It scores each on memorability, spellability, brandability, SEO potential, and domain availability, checks TLD and social media handle availability, and delivers a prioritized top-10 shortlist with detailed rationale and next-steps guidance.

Output: A structured naming report with scoring matrix, top-10 recommendations, availability analysis, social handle strategy, and decision framework.

## Who It's For

- **Startup founders** choosing their company or product name
- **Product managers** naming new features or sub-products
- **Brand strategists** leading rebranding initiatives
- **Marketing leaders** coordinating naming for market entry or repositioning
- **Executive teams** evaluating naming options for stakeholder approval

## Use Cases

- **Pre-launch naming**: Generate domain options before official product launch
- **Rebranding**: Create naming candidates for company rebrand or pivot
- **Sub-product naming**: Name new features or product lines within existing brand
- **Market entry**: Localized domain strategy for new geographic regions
- **Workshop facilitation**: Use naming framework to workshop names with cross-functional teams
- **Social handle strategy**: Ensure consistent brand identity across all digital channels

## Setup

1. **Define product/brand context**: What does it do? Who's it for? What's the brand personality?
2. **Set naming constraints**: Preferred length, TLD, budget, spelling complexity, naming style
3. **Identify decision-makers**: Who approves the final name? When do you need a decision?
4. **Gather stakeholder input**: Collect preferences on naming approaches (real words, invented, metaphors, etc.)
5. **Plan user testing**: If time permits, run quick polls with target users on top 5 candidates

## What You Get

- **20–30 domain name candidates** across eight naming strategies
- **Scoring matrix** rating each on memorability, spellability, brandability, SEO, and availability (1–5 scale)
- **Top-10 ranked shortlist** with detailed rationale for each
- **Availability analysis** showing .com status, alternative TLDs, pricing, and trademark risk
- **Social handle strategy** with handles to reserve and platform priorities
- **Decision framework** explaining how to choose among top 3
- **Backup options** if top choices unavailable or test poorly
- **Next steps** with checklist for domain purchase, email setup, and brand launch

## Common Workflows

- **Startup naming sprint**: Founder provides a 2-sentence product description and audience — skill generates 30 candidates, scores them, and delivers a top-10 shortlist in one session
- **Rebranding workshop**: Use the scoring matrix as a facilitation tool in team workshops — each stakeholder scores independently, then compare
- **Feature naming**: When launching a new product line or sub-feature, run a quick naming round to ensure consistency with the parent brand
- **International expansion**: Generate localized domain strategies when entering new markets — check pronunciation, cultural connotations, and TLD availability per region

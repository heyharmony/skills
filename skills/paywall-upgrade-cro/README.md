# Paywall & Upgrade Screen Optimization

## Overview
A skill for designing in-app paywalls, feature gates, and upgrade screens that convert free users to paid plans. Grounded in customer conversations, product analytics, and pricing psychology.

## What This Skill Does
- Maps trigger points where free users encounter limits (feature gates, trial expiration, usage caps, time-based upsells)
- Extracts value signals and upgrade objections from customer conversations in Harmony
- Designs paywall structure with conversion-optimized copy, feature comparison, and pricing display
- Provides A/B testing and iteration guidance to improve conversion rates
- Handles edge cases: subscribers, trial users, mobile, locales, declining users

## When to Use
- Designing a new paywall or upgrade modal
- Improving paywall conversion rates that are flat or declining
- Creating feature gates that don't annoy users but drive revenue
- Timing upsells to natural product moments (trial expiration, usage limits, feature adoption)
- Running A/B tests on paywall messaging and design

## Key Inputs
- Trigger point (feature gate, usage limit, trial expiration, or time-based)
- Customer conversation data from Harmony (what users value, what objections arise)
- Target segment (free tier, trial, specific cohort)
- Current paywall performance metrics (if optimizing existing paywall)

## Key Outputs
- Paywall specification with headline, value demo, feature comparison, pricing, CTA, and edge cases
- A/B testing plan and success metrics
- Timing and frequency strategy to avoid user frustration

## Connectors
- **HubSpot** — User lifecycle data, upgrade conversion metrics by segment
- **Slack** — Team discussions about upgrade friction and user feedback
- **Stripe** — Payment data, subscription metrics, MRR impact
- **Notion** — Product requirements, pricing documentation, feature roadmap
- **Intercom** — In-app messages, user segments, prompt analytics

## Example
Deskflow (project management SaaS) creates a feature-gate paywall when free users click the Timeline button. Headline: "See your whole project at a glance." Feature comparison shows timeline + integrations as Pro-only. CTA: "Try Timeline for 7 days free." Targets 22% CTR and 6% monthly conversion rate.

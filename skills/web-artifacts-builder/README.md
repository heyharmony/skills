# Web Artifacts Builder

> From concept to interactive tool in one conversation—no build server, no deployment hassle

## What It Does

Turn your business ideas into production-grade web dashboards, data visualizations, and internal tools that work immediately. You describe what you need, and this skill builds a self-contained HTML file using React, TypeScript, and Tailwind CSS. Everything compiles into a single file—open it in any browser, share it via email or link, use it offline. Perfect for OKR trackers, KPI dashboards, client reports, survey tools, and decision-support interfaces.

## Who It's For

- **Product Managers:** Build interactive feature roadmaps, release notes, and capability showcases without waiting for engineering
- **Finance & Operations:** Create real-time budget trackers, expense dashboards, and variance analyzers for board reviews
- **Sales Leaders:** Design client-facing proposal tools, ROI calculators, and capability summaries that close deals faster
- **HR & People Ops:** Build interview feedback tools, compensation planning interfaces, and org chart visualizers
- **Executives & Strategists:** Develop OKR trackers, strategic planning dashboards, and scenario modeling tools for leadership reviews
- **Consultants & Agencies:** Create client dashboards, project trackers, and interactive reports that demonstrate thought leadership

## Use Cases

- **KPI Dashboard for Leadership:** Build a real-time tracker showing sales pipeline, revenue forecasts, and headcount across departments. Before: executives wait for monthly reports. After: live data at your fingertips, filtered by team or metric.

- **Quarterly OKR Planner:** Design an interactive interface where teams set objectives, track key results with color-coded progress, and share status updates. Before: spreadsheets passed via email. After: single source of truth with visual progress indicators.

- **Client ROI Proposal Tool:** Create an interactive calculator that shows projected return on investment, cost savings, and timeline. Before: static PDF presentations. After: clients input their metrics and see personalized outcomes in real-time.

- **Hiring Feedback Portal:** Build a form and dashboard where interviewers submit structured feedback (scores, assessments, notes) and hiring managers see aggregated candidate profiles. Before: scattered emails and spreadsheets. After: standardized data, easy comparison.

- **Marketing Campaign Tracker:** Design a dashboard showing ad spend, impressions, clicks, conversions, and ROI across channels. Before: manual updates in spreadsheet. After: interactive filtering, trend analysis, and instant insights.

- **Expense & Budget Planner:** Build a tool for teams to submit expenses, categorize spending, compare against budget, and forecast quarterly costs. Before: complicated spreadsheets. After: intuitive interface with real-time aggregate views.

## Setup

1. Open Harmony (or use any chat) and describe the tool you need: "I need an OKR tracker for Q1 that shows progress by department"
2. Wait for the HTML file to be generated
3. Click the link or download the file
4. Open in any browser—that's it. No installation, no server, no build process.
5. Share the file with your team via email, Slack, or link

If you need updates later, just ask: "Update the dashboard to add a 'Technology' department" and get a new version.

## What You Get

A fully functional, self-contained HTML file that runs in any modern browser. The artifact includes your custom design (colors, typography, layout), interactive elements (filters, real-time updates, dynamic forms), and responsive behavior across mobile, tablet, and desktop. Everything is optimized for performance—typically 100-500 KB depending on complexity. You can share, iterate, and evolve it without any technical setup.

# Set Up Analytics Tracking

Build a measurement strategy tied to your business goals, then implement GA4, Google Tag Manager, and event tracking with clean naming conventions, UTM strategy, and privacy safeguards.

## Quick Overview

This skill walks you through:

- **Defining what to track:** Extract business goals from team discussions (planning calls, leadership meetings, product strategy). Translate these into conversion events that matter.
- **GA4 setup:** Create properties, data streams, consent mode, and conversion events. Avoid data pollution; keep dev/prod separate.
- **Google Tag Manager:** Build a scalable tag container. Centralize tracking so non-engineers can add events without touching code.
- **Event naming & data dictionary:** Use consistent naming (snake_case, specific), document all properties and allowed values.
- **UTM strategy:** Define naming templates for paid, organic, email, internal campaigns. Automate generation to prevent typos.
- **Data quality & privacy:** Implement validation rules, set alerts, ensure GDPR/CCPA compliance.

## When to Use

- Starting a GA4 migration or new property setup
- Redesigning your tracking architecture (too many custom events, inconsistent naming)
- Building a measurement plan from scratch (new product line, campaign, website)
- Debugging tracking issues (events not firing, UTMs broken, data gaps)
- Auditing privacy compliance (GDPR, CCPA, data retention)

## What You'll Get

A **Tracking Implementation Plan** that includes:
- Business goals → conversion events mapping
- GA4 property & GTM container checklists
- Event naming conventions & data dictionary
- UTM strategy with templates
- Data quality rules & validation
- Privacy compliance checklist
- Success metrics & 90-day timeline

## Key Inputs

- Business goals & KPIs (from team conversations, strategy docs)
- Funnel definitions (from HubSpot, CRM, or product requirements)
- Product success metrics (from Notion, feature specs)
- Current tracking setup (if migrating or auditing)

30–60 minutes to plan; 1–2 weeks to deploy GA4 + GTM + validate data.

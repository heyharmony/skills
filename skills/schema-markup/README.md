# Schema Markup Implementation

> **Add structured data (JSON-LD) to your website so Google surfaces rich results, knowledge panels, and star ratings in search—plus help AI systems understand your content better.**

## What It Does

This skill guides you through implementing schema.org markup (JSON-LD format) on your website pages. It helps you:

- Generate valid JSON-LD blocks for Organization, Product, Article, FAQ, Event, LocalBusiness, HowTo, and BreadcrumbList schemas
- Enable Google rich results: star ratings, pricing, availability, FAQ snippets, breadcrumb trails
- Improve AI content understanding by providing explicit semantic structure
- Validate your markup using Google Rich Results Test and Schema.org validators
- Implement schemas across your tech stack (static HTML, React/Next.js, WordPress, etc.)
- Monitor indexed markup in Google Search Console

**Powered by your Harmony data**: The skill can pull product details from HubSpot, FAQ patterns from Slack conversations, content from Google Drive, and documentation from Notion—so your schema reflects real customer discussions and business logic.

## Setup

1. Open this skill and have your website URL ready
2. (Optional) Connect Google Drive, Slack, HubSpot, and/or Notion to auto-populate schema data
3. List your key page types (homepage, product, article, event, FAQ, etc.)
4. Run the skill to generate JSON-LD for each
5. Implement the markup in your tech stack (see implementation guidance in skill)
6. Validate with Google Rich Results Test
7. Deploy and monitor in Google Search Console

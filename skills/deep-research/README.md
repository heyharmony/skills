# Deep Research Report

A skill for producing comprehensive, structured research reports on any topic.

## What It Does

Takes a topic and produces a well-sourced research report with executive summary, key findings, multiple perspectives, implications, and recommended actions. Reports scale from a quick 1-page overview to a 10+ page deep dive depending on what the user needs.

## When It Triggers

- "Research [topic] for me"
- "Do a deep dive on [topic]"
- "I need to understand [topic]"
- "Put together a report on [topic]"
- "What's the full picture on [topic]?"
- Any request for standalone research not tied to a specific meeting or conversation

## Output

A markdown file saved to the user's workspace, structured with:

- Executive Summary (key findings + bottom line)
- Background & Context
- Current State
- Key Findings (with evidence and sources)
- Different Perspectives
- Implications (risks + opportunities)
- Recommended Actions
- Sources (with reliability notes)
- Questions for Further Research

## Depth Levels

| Level | Length | Use Case |
|-------|--------|----------|
| Quick overview | ~1 page | Get oriented fast |
| Standard | 3–5 pages | Default — solid coverage with evidence |
| Deep dive | 10+ pages | Comprehensive treatment for major decisions |

# Value Proposition

> Articulate exactly why customers should choose you — in a format that powers every decision downstream.

## What It Does

Builds a detailed value proposition using a 6-part JTBD template: Who, Why (Problem), What Before, How (Solution), What After, and Alternatives. Produces a structured analysis plus a ready-to-use Value Proposition Statement and Positioning Statement. When conversation data is available, grounds everything in the language customers actually use to describe their problems and desired outcomes.

## Who It's For

Product managers, founders, and marketers who need a clear, defensible value proposition — whether for a new product, a repositioning effort, or as input to messaging and strategy work.

## Use Cases

- Defining a new product's value proposition → get a complete 6-part analysis with competitive alternatives mapped
- Repositioning after customer discovery → value prop grounded in real JTBD language from Harmony conversations
- Feeding downstream work → the Value Proposition Statement plugs directly into the Value Proposition Statements skill for marketing/sales/onboarding copy

## Setup

Install and go. Provide the product and target audience. For best results, include competitive context and point to Harmony conversations (customer interviews, sales calls, churn debriefs) for authentic customer language.

## What You Get

A structured 6-part value proposition (Who, Why, What Before, How, What After, Alternatives), a concise Value Proposition Statement, a marketing-ready Positioning Statement, and optionally a competitive Value Curve.

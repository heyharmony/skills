# Slack GIF Creator

> Turn team moments into custom animated reactions that stick and spark joy

## What It Does

Create custom animated GIFs optimized for Slack and messaging platforms. Design celebratory animations, emoji reactions, status indicators, and visual responses that bring personality to team communication. This skill generates polished, loop-ready GIFs (128x128 for emoji reactions, 480x480 for message GIFs) with smooth animations, intentional color palettes, and strategic timing. Upload to Slack's emoji library or share as message reactions that teammates use again and again.

## Use Cases

- **Ship Celebration GIF:** Create an animated sailing ship, rocket, or confetti burst that triggers when the team ships a feature. Before: "great work everyone!" text. After: everyone reacting with a custom :ship: emoji that feels like genuine celebration.

- **Deal Closed Animation:** Build a GIF of fireworks, a trophy, or a smiley face celebrating, that the sales team uses when they close a deal. Before: boring typed +1 reactions. After: visual celebration that sparks energy and momentum.

- **Status Indicator GIFs:** Design a set of small emoji reactions (focus mode, in a meeting, on PTO, grab coffee) that teammates use to update availability without interrupting. Before: manual status updates in bio. After: fun visual context.

- **Team Mascot Animation:** Create a branded character or logo that moves, waves, or reacts in a loop. Use across all team channels as your visual brand. Before: static logo. After: an animated personality that represents your team.

- **Approval Workflow GIF:** Design a "approved" or "needs review" animation that code reviewers use when responding to pull requests. Before: "+1" comments. After: a visual, trackable reaction that adds levity to code review.

- **Holiday/Seasonal GIFs:** Build a small library of themed animations (holiday lights, falling snow, spring growth, spooky reactions) that appear seasonally in #general. Before: no seasonal context. After: visual reminders that the team celebrates together throughout the year.

## Setup

1. In Harmony (or chat), describe the animation you want: "Create a celebratory ship GIF for when we ship features"
2. Share any team colors, tone preferences, or reference animations you like
3. Wait for the GIF file to be generated
4. Download the GIF
5. Upload to Slack: Admin Panel > Emoji > Custom Emoji > Upload file
6. Rename (e.g., :ship-celebration:) and save
7. Share with your team and watch it get used

## What You Get

A production-ready, loop-optimized GIF file at your chosen dimensions (128x128 for emoji, 480x480 for larger reactions). Includes smooth frame-by-frame animation with intentional easing and timing, a cohesive color palette that works on light and dark Slack themes, and file optimization for fast loading (typically 100–500 KB). The animation is designed to loop seamlessly and look great when reused dozens of times daily. Source details (frame count, timing, easing) included so you can create variations later.

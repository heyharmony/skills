# Translation

## Description

Translates text between languages while preserving tone, context, and formatting.

## Triggers

- Manual invocation

## Inputs

| Name   | Type   | Required | Description                        |
| ------ | ------ | -------- | ---------------------------------- |
| text   | string | yes      | Text to translate                  |
| target | string | yes      | Target language                    |
| tone   | string | no       | Desired tone (formal, casual, etc.)|

## Outputs

Translated text in the target language, maintaining the original structure and intent.

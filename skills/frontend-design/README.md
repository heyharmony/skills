# Frontend Design

> Ship distinctive, production-grade web interfaces that feel intentional, not templated

## What It Does

Build beautiful, custom web interfaces and landing pages with intentional design decisions—distinctive typography, cohesive color systems, strategic animations, and unconventional spatial composition. This skill establishes clear aesthetic direction first, then executes in semantic HTML, modern CSS, and vanilla JavaScript. You get production-ready frontend code that stands out from template-generated websites and looks thoughtfully designed. Perfect for landing pages, internal tool UIs, marketing microsites, admin dashboards, and interactive prototypes.

## Setup

1. In Harmony (or chat), describe the site you want and its purpose: "I need a landing page for my SaaS that positions us as the thoughtful alternative to competitors"
2. Share your aesthetic direction, brand colors, or reference sites you admire
3. Wait for the HTML, CSS, and JavaScript files to be generated
4. Download the files and deploy to your hosting (Vercel, Netlify, GitHub Pages)
5. Customize content as needed
6. Launch and share with the world

Hosting integration is straightforward—most modern hosts auto-detect and deploy static HTML projects in seconds.

## What You Get

Clean, semantic HTML with CSS variables for easy customization, a cohesive color system, responsive design for all screen sizes, and subtle animations that enhance usability. The code is production-ready, accessible (WCAG AA compliant), and optimized for performance. Includes documentation on typography choices, color intent, and animation principles so you can maintain consistency as it evolves. No frameworks required—pure HTML, CSS, and vanilla JavaScript.

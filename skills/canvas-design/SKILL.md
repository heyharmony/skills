## Purpose

Transform complex information and strategic ideas into pristine, museum-quality visual artifacts. This skill articulates a clear aesthetic philosophy first—establishing intent around form, space, color, and composition—then manifests that vision as a design that communicates through visual language rather than words. Produces publication-ready PDFs or PNGs for executive presentations, brand identity systems, thought leadership pieces, and visual strategy documents that stand out from generic templates.

## When to Use

- Designing a premium one-pager that communicates your company's strategic differentiation to investors or enterprise clients
- Creating a visual report that presents quarterly results, market analysis, or transformation initiatives to leadership
- Building a brand mood board or design system visual that codifies aesthetic identity for a new company or product
- Designing an infographic that explains a complex methodology, process, or data story in pure visual terms
- Creating a thought leadership poster or visual manifesto that articulates a movement or paradigm shift
- Designing visual identities, color palettes, or typography systems that guide all future design work
- Building presentation slides or conference materials that elevate a talk above standard templates

## Instructions

### 1. Articulate Design Philosophy as Manifesto

- Define the aesthetic movement or vision in 4-6 paragraphs: What ideas drive this design? What's the emotional tone?
- Establish principles around form: Is the design geometric/organic? Structured/flowing? Balanced/asymmetrical?
- Clarify spatial logic: How does whitespace create meaning? What's the relationship between elements and emptiness?
- Define the color strategy: What palette tells this story? What's the emotional or conceptual logic behind each hue?
- Set typography intent: What voices or moods do typefaces establish? How do font sizes create hierarchy and emphasis?
- Document composition rules: What's off-limits? What makes this design distinctly *this* vision, not another?

### 2. Research Reference Aesthetics

- Study 3-5 design movements, eras, or artists that align with your manifesto (mid-century modernism, constructivism, bauhaus, contemporary minimalism, etc.)
- Identify visual patterns: color relationships, spatial proportions, typographic rhythms, material textures
- Collect inspiration for compositional approaches: How do master designers use grid, asymmetry, and visual flow?
- Note what NOT to do: Identify clichés or overused patterns your design will consciously avoid
- Extract principles: Distill specific rules (grid systems, color combos, spatial ratios) to guide execution

### 3. Establish Grid and Spatial Hierarchy

- Choose a grid system: Standard 12-column grid, golden ratio, modular square grids, or custom proportions
- Map out primary zones: Header, content area, footer, margins, breathing room
- Define whitespace language: How much empty space creates elegance vs. emptiness? What's your ratio?
- Establish alignment: Every element should follow grid or intentional asymmetry—never random placement
- Create visual weight distribution: Balance mass, color intensity, and focal points across the canvas

### 4. Select and Compose Color Palette

- Start with 2-3 primary colors that embody the core idea: What's the dominant tone?
- Add 2-3 secondary colors for accent and hierarchy: Which colors create visual interest without noise?
- Define neutral tones: Backgrounds, text, and border colors that let primary colors shine
- Test color relationships: Do they have conceptual logic? Do they feel intentional or arbitrary?
- Build contrast matrix: Ensure readability between text and backgrounds; test for accessibility (WCAG compliance)
- Lock the palette: Every color choice should feel inevitable, not interchangeable

### 5. Design with Intentional Typography

- Select 2 typefaces maximum: A primary typeface for body/labels and a distinctive typeface for headlines or emphasis
- Establish font-size hierarchy: Define exact pixel sizes for H1, H2, body text, captions (e.g., 72px headline, 32px subhead, 14px body)
- Use weight strategically: Bold conveys importance; regular is neutral; light adds sophistication
- Create rhythm through spacing: Consistent line-height, letter-spacing, and margins make typography feel composed
- Avoid common AI defaults: Skip "Inter", "Poppins", "sans-serif". Choose typefaces with distinctive character
- Test for legibility: Ensure sufficient contrast and size for print (8pt minimum for small text)

### 6. Build Visual Composition

- Place the primary focal point: What's the visual anchor that draws the eye first?
- Develop secondary focal points: Where does the eye move next? Create a visual journey
- Use asymmetrical balance: Offset visual weight creates tension and sophistication (vs. centered monotony)
- Integrate imagery strategically: Abstract shapes, photography, or illustration should support the narrative—not decorate
- Layer information: Foreground elements, mid-ground content, background context—create depth through spatial relationships
- Eliminate clutter: Every element should earn its place; remove decorative elements that don't communicate
- Test negative space: Can you remove anything? The strongest designs say more with less

### 7. Produce Polished Artifact (PDF or PNG)

- Build design in professional tool (Adobe InDesign, Figma, or equivalent)
- Set proper export settings: PDF/X-1a for print (300 DPI), PNG at 2x resolution for screen (96-144 DPI)
- Verify color management: RGB for digital, CMYK for print; ensure no overlapping elements or unintended artifacts
- Check alignment: No rough edges, misaligned text, or stray pixels. Zoom to 100% and verify crisp rendering
- Include metadata: Embed title, version, date, and creator info in file properties
- Create print specs if physical: Trim marks, bleed area, margin notes—all standard production requirements
- Final review: Print proof or screen-test at actual size; adjust for any rendering surprises

```

- Resolution: 300 DPI (print) or 144 DPI (digital)
- Color Space: CMYK (print) or RGB (digital)
- Bleed: [0.125"–0.25" if printing]

Composition:
- Dominant focal point: [Description of visual anchor and emotional weight]
- Color palette: [Hex or CMYK values for each color with usage intent]
- Typography: [Typeface names, sizes, weights used]
- Grid structure: [Column count, margin widths, element alignment rules]

Visual Elements:
- Background: [Solid, gradient, texture—with intent]
- Primary content: [Imagery, illustration, or data visualization—with spatial logic]
- Typography blocks: [Headlines, body copy, annotations with hierarchy]
- Accent elements: [Borders, shapes, dividers that organize space]

Narrative Arc:
[1-2 sentence description of how the eye moves through the design and what story emerges]
```

## Example

**Input Scenario:** Create a visual manifesto for "Intentional Design"—a thought leadership piece for a design consultancy that challenges the prevalence of AI-generated templates and advocates for purposeful, distinctive work.

**Design Name:** intentional-design-manifesto.pdf

**Design Philosophy (Manifesto):**

"Intentional Design rejects the false efficiency of algorithmic templates and template-based thinking. It demands deliberation at every decision point—why this color, not another? Why asymmetry here, balance there? Form follows purpose, not precedent. We believe in the discipline of constraint: limited palettes, minimal typefaces, generous whitespace. Visual clarity emerges not from decoration but from the ruthless elimination of the unnecessary. Every element earns its presence through functional or conceptual logic. The result is artifacts that feel inevitable, not arbitrary—designs that could only exist this way."

**Visual Execution:**

```

Resolution: 300 DPI (print-ready)
Color Space: CMYK

VISUAL STRUCTURE:

Background:
- Rich, near-black charcoal (CMYK: 70, 60, 60, 70) fills 60% of composition
- Upper right: pure white (CMYK: 0, 0, 0, 0) breathing room
- Subtle texture (300-pixel repeating dot pattern) at 15% opacity adds tactile quality

Primary Headline (Left third, centered vertically):
- Typeface: Saol Text Bold (serif, high contrast, distinctive)
- Size: 96pt, color: pure white
- Text: "INTENTIONAL"
- Spacing: 120% line-height, tight letter-spacing (-0.02em)
- Visual weight: Heavy, anchors the entire composition

Secondary Headline (Below, continuing left column):
- Typeface: Maax Light (sans-serif, geometric, minimal)
- Size: 42pt, color: accent gold (CMYK: 0, 20, 100, 0)
- Text: "DESIGN"
- Spacing: 2em below primary headline
- Visual effect: Creates vertical rhythm and separates concepts

Content Block (Right third, stacked narrative):
- Background: white rectangle, 0.25" margin from all edges
- Typeface: IBM Plex Serif Regular (readable, authoritative, not trendy)
- Size: 12pt body, 16pt for emphasis, 10pt for captions
- Color: near-black (CMYK: 0, 0, 0, 95)
- Line-height: 1.6em (generous breathing room)
- Width: 350px (optimal reading length)

Manifesto Text (excerpt, left-aligned within white block):

"We reject templates. Every decision—color, space, form—serves the story. Design is not decoration. It is strategy made visible."

- Key phrase highlighted in gold: "serves the story"
- 2pt gold underline (0.125" below text)
- Small caps for opening phrase: "WE REJECT"

Accent Elements:
- Single vertical line (2pt, gold, full height) separates left narrative from right content
- Three evenly spaced circles (1", 0.75", 0.5" diameter) stacked on far left; colors: charcoal, gold, white (creates visual cadence)
- Footer: Small text (8pt, gray) reads "INTENTIONAL DESIGN MANIFESTO | V1.0 | MARCH 2026"

Spatial Logic:
- 3 inches of white space on right edge: silence creates emphasis on content
- Left margin: 1 inch; right margin: 3 inches (intentional asymmetry conveys weight to narrative)
- Golden ratio proportions: 1.618 ratio between headline size and body text size
- Content hierarchy: Large headline dominates; manifesto text whispers narrative; footer anchors with date

Color Palette (locked):
- Near-black (CMYK: 70, 60, 60, 70) — authoritative, timeless
- Pure white (CMYK: 0, 0, 0, 0) — clarity, space
- Gold (CMYK: 0, 20, 100, 0) — accent, intentionality, warmth

Typography Hierarchy:
- H1 (Saol Bold, 96pt): Dominance, visual anchor
- H2 (Maax Light, 42pt): Accent, complement to H1
- Body (IBM Plex Serif, 12pt): Narrative voice, readability

- 11" x 17" PDF, 300 DPI, CMYK color space
- File size: ~8 MB (includes texture and high-quality assets)
- Print-ready: Includes 0.125" bleed, crop marks, color bar
- Accessible: All text embedded as fonts; color contrast ratio 7:1 (exceeds WCAG AA)
```

## Edge Cases

- **Multi-language layouts:** If the design needs to accommodate different text lengths (e.g., English vs. German), test reflow at different language settings. Consider fixed-width regions vs. dynamic text wrapping.
- **Print vs. digital adaptation:** Design may render differently in print vs. screen. Test both CMYK (print) and RGB (digital). Verify color shifts; adjust saturation if needed for accuracy.
- **Accessibility requirements:** Ensure color contrast ratios meet WCAG standards (4.5:1 for text, 3:1 for graphics). Don't rely on color alone to convey meaning—use patterns, icons, or text labels as backup.
- **Resolution quality on screen:** If PNG at 144 DPI looks soft on high-DPI displays, export at 2x (288 DPI) and resize in web. Verify no pixelation or anti-aliasing artifacts.
- **Complex data visualization:** If infographic includes charts or data, ensure visual clarity—labels must be readable at distance; color palette must work for colorblind viewers; data shouldn't be obscured by decorative elements.
- **Iterative feedback:** Design may need tweaking after client review. Lock dimensions, grid, and color palette; iterate on copy placement, accent emphasis, or spacing adjustments—not fundamental restructuring.

Now what? Present the artifact to stakeholders, gather feedback on visual clarity and narrative impact, iterate on copy or composition if needed, and prepare for production (print, web, presentation deck integration).
